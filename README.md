# Krishi Sakhi 🌱

Krishi Sakhi is an AI-powered personal assistant for farmers that provides weather updates, crop disease detection, mandi rates, and government scheme information.

## Features

- Weather Forecast
- Crop Disease Detection
- Mandi Rates
- Government Schemes
- AI Chat Assistant

## Technologies Used

- HTML
- CSS
- JavaScript
